# QDtb Neovim configuration documentation

This repository contains the documentation for my personal Neovim configuration.

> [!NOTE]
> The QDtb repository is [github.com/natebass/qdtb](https://github.com/natebass/qdtb).
> The code should be periodically kept up to date with the main repository by manually copy/pasting the `qdtb/` folder.

## Prerequisites

- lua
- luarocks
- ldoc
- PNPM

## Get started

1. `pnpm i`
2. `pnpm build-plugin`
3. `pnpm serve`
