# Fish Shell Configuration

Welcome to the Fish shell documentation section. Here you can find information about my Fish setup, functions, and abbreviations.

## Features

- **Friendly Interactive Shell**: Auto-suggestions and syntax highlighting.
- **Custom Functions**: Located in `~/.config/fish/functions/`.
- **Abbreviations**: Quick shortcuts for common commands.

## Installation

On Linux Mint:

```bash
sudo apt install fish
```

## Setting as Default

```bash
chsh -s /usr/bin/fish
```
