# QDtb Neovim Configuration

This project is meant to be installed in the root of your Neovim config folder. For common Linux XDG folders, see [the PowerShell XDG documentation](/docs/powershell/xdg-setup).

## Neovim (or Neovide) with Flatpak

Instead of the normal distribution installation, I want to focus on Vim configuration with Flatpak. I find it easier to install the latest version of Neovim with Flatpak, but it comes with some different default paths.
